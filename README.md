# ChronoLearn Evaluation Survey

This repository hosts the official evaluation survey for the **ChronoLearn** graduation project.

ChronoLearn is an AI-powered educational platform that employs Knowledge Graphs and Large Language Models to enhance the learning of Jordanian and Palestinian history. The purpose of this survey is to **assess the system’s usability, educational effectiveness, feature quality, and overall user experience** as part of the project’s academic evaluation. 
 
 This survey represents the evaluation component of the main ChronoLearn project available here: 
 
 👉 https://github.com/Mohammad-ALADDASI/ChronoLearn

## Purpose of This Survey

The survey is designed to:

* Evaluate **user interaction and usability**
* Measure **learning effectiveness compared to traditional methods**
* Assess **AI-driven features and knowledge graph components**
* Collect **user feedback for system improvement**

This directly supports the research objectives and evaluation phase of the ChronoLearn system.
